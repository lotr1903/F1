# F1 APP

This project simpley fetches the F1 champions between 2005 and 2015. 
You can also check each season race results separately by clicking each item in the list.
Items behaves as accordions so each season's race results will appear underneath of related item.

## Getting Started

### Prerequisites
This project uses tools and frameworks such as

* React +16
* Webpack 4
* Babel 7
* Axios
* Sass
* Jest
* Enzyme
* Eslint/Sasslint/Prettier


## Installing
To install the project run below command in project folder.

- `npm install`

## Running the project
To check the project in your local machine run below command in project folder

- `npm start`

When webpack build finishes it will automatically open browser and run the project in 

[localhost:3000](http://localhost:3000)

## Running the tests
To see tests simply run below command in project folder.

- `npm test`

If you want run test during adding new tests run below command 

- `npm run test:watch`

If you want to see project's test coverage results run below command

- `npm run test:coverage`

## Deployment
To prepare project for production run below command in project folder.

- `npm run prod`

When webpack finished you can use /dist folder in root to deploy it.
